# BMI Calculator (React)

A modern BMI Calculator built using ReactJS with a clean and responsive UI.

## Features
- Dark mode toggle
- Metric & Imperial unit conversion
- Health tips based on BMI
- Responsive design

## Tech Stack
- ReactJS
- JavaScript
- CSS

## Live Demo
https://sakshi-bmi-calculator.netlify.app

## Run Locally
npm install  
npm start
